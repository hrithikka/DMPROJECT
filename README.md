# DMPROJECT
MILESTONE-1
Installing WSL 2
->Opening Windows Command Prompt in administrator mode by right-clicking and selecting "Run as administrator",
then entered the wsl --install command, then restarted my machine.
->Ensuring that the distribution runs in WSL 2 mode

Installing DOCKER DESKTOP
->Downloaded the Docker "Desktop Installer.exe" from the official website.
->When prompted, ensured WSL 2 option on the Configuration page is selected.
->Followed the instructions on the installation wizard to authorize the installer and proceeded with the installation.
->Docker successfully installed and Docker shoetcut is added to the desktop.
->When Docker Desktop starts, Settings > Resources > WSL Integration->On UBUNTU toggle-> Apply & Restart.

VS Code Remote Development
->Open VS code
->Installed  Remote Development extension pack
